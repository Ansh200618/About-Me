# 🚀 Quick Deployment Guide

## Deploy in 5 Minutes with GitHub Pages

### Step 1: Enable GitHub Pages
1. Go to your repository on GitHub: `https://github.com/Ansh200618/About-Me`
2. Click on **Settings** (top menu)
3. Scroll down to **Pages** (left sidebar)
4. Under "Build and deployment":
   - **Source**: Select "Deploy from a branch"
   - **Branch**: Select "main" (or your current branch)
   - **Folder**: Select "/ (root)"
5. Click **Save**

### Step 2: Wait for Deployment
- GitHub will automatically deploy your site
- This takes 2-3 minutes
- A green checkmark will appear when ready

### Step 3: Access Your Portfolio
Your portfolio will be live at:
```
https://ansh200618.github.io/About-Me/
```

## Testing Locally

Before deploying, you can test your portfolio locally:

1. Simply open `index.html` in your web browser
2. Or use a local server (optional):
   ```bash
   # If you have Python installed:
   python -m http.server 8000
   
   # Then visit: http://localhost:8000
   ```

## Updating Your Portfolio

After making changes:
1. Commit your changes to GitHub
2. Push to the branch you configured in GitHub Pages
3. GitHub will automatically redeploy (2-3 minutes)
